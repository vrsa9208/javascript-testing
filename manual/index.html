Javascript Testing

Introducción

El desarrollo de tests es una parte fundamental en el proceso de desarrollo de software. Esto es así para cualquier lenguaje de programación, sea C#, Java, Javascript, etc. En el ámbito de Javascript contamos con multiples opciones para realizar dichas pruebas. En este laboratorio nos estaremos enfocando en 2 de las tecnologías más populares:

Jasmine

Es un framework de desarrollo orientado al comportamiento para realizar pruebas en Javascript. No depende de ningún otro framework de Javascript. No requiere un DOM (Document Object Model). Y cuenta con una sintaxis limpia y sencilla para poder escribir facilmente los test.

Visitar el sitio de Jasmine

Karma

El objetivo principal de Karma es proveer de un entorno de pruebas para desarrolladores. Un entorno donde no se tengan que preparar tantas configuraciones y se tenga el enfoque mejor en escribir el código y recibir una retroalimentación instantanea de los test.
